# DagangPlay

Project structure:
- `frontend`: Next.js 14+ (App Router)
- `backend`: NestJS + Prisma ORM

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Configure your `.env` with `DATABASE_URL`
4. `npx prisma generate`
5. `npm run start:dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
